This project demonstrates the end-to-end analysis of an e-commerce company’s user activity logs to extract actionable business insights. The main objectives were to build a conversion funnel, prepare data for cohort analysis, and calculate customer retention rates.

Key Highlights:
	•	Conversion Funnel: Tracked how users progress from product page views to purchases, calculating total conversion rates and step-by-step conversion rates.
	•	Cohort Analysis Preparation: Filtered purchase events, determined each user’s first purchase date, and organized transactions by event month and cohort age.
	•	Retention Rate Calculation: Aggregated purchase data into monthly cohorts and calculated retention rates over a 4-month period.
	•	Data Documentation & Organization: Created a polished spreadsheet with an executive summary, table of contents, clear formatting, and properly labeled sheets to communicate results effectively to stakeholders.

This project showcases advanced spreadsheet techniques, pivot table analysis, and data-driven decision-making for e-commerce performance insights.
