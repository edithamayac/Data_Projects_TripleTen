# E-COMMERCE COMPANY ANALYSIS

## Project Description

This project demonstrates the end-to-end analysis of an e-commerce company’s user activity logs to extract actionable business insights. The main objectives were to build a conversion funnel, prepare data for cohort analysis, and calculate customer retention rates.

## Tools Used 

- MS Excel

## Key Findings

- 29% of the total of people went from viewing to putting in the shopping cart, and from there 36% proceeded to paid for it. Which tells us only 10% of the total of people went from viewing to buying.
- Retention rates significantly declined across cohorts after the first purchase month
